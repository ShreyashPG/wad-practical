B.E. Computer Engineering

  • Course Structure

First Year (FE)

  • Engineering Mathematics-I

    Differential calculus, matrices, and numerical methods

  • Basic Electrical Engineering

    Fundamentals of DC/AC circuits and electrical machines

  • Programming in C

    Structured programming concepts and problem solving

Second Year (SE)

  • Data Structures

    Algorithms, trees, graphs, and complexity analysis

  • Computer Organization

    CPU architecture, memory hierarchy, and I/O systems

  • Discrete Mathematics

    Logic, combinatorics, and graph theory fundamentals

Third Year (TE)

  • Operating Systems

    Process management, memory allocation, file systems

  • Database Management Systems

    Relational algebra, SQL, and normalization techniques

  • Computer Networks

    OSI model, TCP/IP, and network protocols

Final Year (BE)

  • Machine Learning

    Supervised/unsupervised learning algorithms

  • Cloud Computing

    Virtualization, SaaS, PaaS, and cloud security

  • Big Data Analytics

    Hadoop ecosystem and data processing techniques
